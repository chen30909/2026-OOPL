# Game Introduction
### 遊戲內容與簡介
一款 2D 的槍枝攻擊涉及遊戲，具備豐富的怪物種類與刺激的戰鬥關卡，多樣性的互動提供新鮮感，是個奇幻風冒險單機遊戲。

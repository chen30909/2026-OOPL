# Game Introduction

VVVVVV Demo 是經典 2D 復古平台益智遊戲的免費體驗版。玩家扮演船長 Viridian，無法跳躍，只能透過「翻轉重力」來操控角色在天花板與地面間穿梭，挑戰快速、高難度且極簡的關卡設計。 

[遊戲連結](https://www.youtube.com/watch?v=jkUK8B6LiuU)
